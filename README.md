# AlgorithmPractice
알고리즘 문제 풀이 저장소

## 사용한 IDE
- IntelliJ IDEA 2020.1.1 (EDU)

## 소스코드 작성법 및 실행방법
- 문제 풀이는 각각의 클래스를 생성해서 작성한다.
- 각각의 문제는 Problem 클래스를 상속받는다. 
- Main.java의 problems 리스트에 클래스를 넣는다.
- 풀이는 solution() 메서드에 작성한다. 
- 실행하고자 하는 문제의 position에서 run()을 호출해 실행한다.
